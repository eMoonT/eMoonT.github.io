    内部函数对外部函数作用域里变量的引用（非全局变量），则称内部函数为闭包

  • time模块

    　在Python中，通常有这几种方式来表示时间：

    • 时间戳(timestamp)：通常来说，时间戳表示的是从1970年1月1日00:00:00开始按秒计算的偏移量。我们运行“type(time.time())”，返回的是float类型。
    • 格式化的时间字符串(Format String)
    • 结构化的时间(struct_time)：struct_time元组共有9个元素共九个元素:(年，月，日，时，分，秒，一年中第几周，一年中第几天，夏令时)

    • 创建数据库

    • 查看当前数据库中所有的表

      格式：show tables;

      说明： auto_increment 自增长 primary key 主键 not null 不为空

      格式：create table 表名(id int auto_increment primary key, name varchar(20) not null, age int not null, gender bit default 1, address varchar(20), isDelete bit default 0);

    • 删除表

      格式：drop table 表名;

    • 查看表结构

      格式：desc 表名；

    • 查看建表语句

      格式：show create table 表名；

    • 重命名表名

      格式：rename table 原表名 to 新表名；

      格式：alter table 表名 add|change|drop (列表 类型)；

  • 数据操作

    • 增

      • 全列插入：

        说明：主键是自动增长的，但是在全列插入时需要占位，通常使用0，插入成功后以实际数据为准

        格式： insert into 表名 values(…)；

        示例：insert into student1 values(0, “习近平”, 56, 1, “北京”, 0);

      • 缺省插入：

        格式：insert into 表名(列1， 列2，…) values(值1， 值2，…);

        示例：insert into student(name, age, address) values(“张”, 19, “上海”);

      • 同时插入多条数据：

        格式：insert into 表名 values(), ();

    • 删

      格式：delete from 表名 where 条件；

      示例：delete from student1 where id=4;

      注意：没有条件时会删除所有数据

    • 改

      格式：update 表名 set 列1=值1，列2= 值2，… where 条件

      示例：update student1 set age=16, where id = 7;

      说明：没有条件时会修改全部数据

    • 查

      说明：查询表中的所有数据

      格式：select * from 表名；

  • 查

    • 基本语法

      说明： from是关键词后面是表名，表示数据来源这张表

      select后面写表中的列名，如果是*表示在结果集中显示表中的所有列

在select 后面的列名部分，可以使用as为列名起别名，这个别名显示在结果集中

如果要查询多个列，之间使用逗号分隔

      格式： select * from 表名；

      示例： select * from student1;

      select name, age, address from student1;

select name as a, age  from student1;

    • 消除重复行

      说明：在select后面列前面使用distinct可以消除重复的行

      示例: select distinct from student1;

      select distinct gender from student1;

    • 条件查询

      • 语法

        select * from 表名 where 条件

      • 比较运算符

        等于 =

        大于 >

        小于 <

        大于等于 >=

        小于等于 <=

        不等于 !=

        需求： 查询id值大于8的所有数据

        示例： select * from student1 where id>4;

      • 逻辑运算符

        and 并且

        or 或者

        not 非

        需求：查询id值大于4的女同学

        示例：select *from student1 where id>4 and gender=0;

      • 模糊查询

        说明：like %表示任意多个任意字符 _表示一个任意字符

        示例：select * from student1 where name like “习%”;

      • 范围查询

        说明：in 表示在非连续的范围内 between…and … 表示在一个连续的范围内

        需求：查询编号为3,5,8,10的学生

        示例：select * from student1 where id in (3, 5, 8, 10);

        需求：查询编号为6到8的学生

        示例：select * from student1 where id between 6 and 8;

      • 空判断

        注意：null 与””是不同的

        判断空：is null

        判断非空： is not null

        需求：查询没有地址的数据

        示例：select * from student1 where address is null;

      • 优先级

        小括号，not 比较运算符，逻辑运算符

        and 比 or 优先级高，如果同时出现希望先选or,需要结合()来使用

    • 聚合

      为了快速等到统计数据，提供了五个聚合函数

      • count(*)

        说明：计算总行数，括号可以写*和列名

        需求：查询学生总数

        示例：select count(*) from student1;

      • max(列)

        说明：求此列的最大值

        需求：查询女生编号的最大值

        示例：select max(id) from student1 where gender=0;

      • min(列)

        需求：查询女生编号的最小值

        示例：select min(id) from student1 where gender=0;

        说明：求此列的最小值

      • sum(列)

        需求：查询所有学生的年龄和

        示例：select sum(age) from student1;

        说明：求此列的和

      • avg(列)

        需求：查询所有学生年龄的平均值

        示例：select avg(age) from student1;

        说明：求此列的平均值

    • 分组

      按照字段分组，表示此字段相同的数据将会被放在一个集合中，

      分组后，只能查询相同的数据列，对于有差异的数据列无法显示在结果中

      可以在分组后的数据进行统计，做聚合运算

      语法：select 列1, 列2, 聚合… from 表名 group by 列1, 列2 , …

      需求：查询男女生总数

      示例： select gender, count(*) from student1 group by gender;

      分组后的数据筛选：select 列1, 列2, 聚合… from 表名 group by 列1, 列2 , … having 列1 列2 聚合

      示例：select gender, count(*) from student1 group by gender having gender;

      where 与having 的区别：

      where 是对from 后面的指定的表进行筛选，属于对原始数据的筛选

      having 是对group by 的结果进行筛选

    • 排序

      语法：select from 表名 order by 列1 asc|desc, 列2 asc|desc, …

      说明： 将数据按照列1进行排序，如果某天列1的值相同则按照列2进行排序，往复循环

      默认按照从小到大的顺序排序

asc升序

desc降序

      需求：将没有被删除的数据按照年龄排序

      示例： select * from student1 where isDelete=0 order by age asc;

      select * from student1  where isDelete=0 order by age desc,id desc;

    • 分页

      语法： select * from 表名 limit start, count;

      说明： start 索引从0开始

      示例： select * from student1 limit 0, 3;

      select * from student1 where gender=0 limit 0, 3;

  • 关联

    建表语句：create table class(id int auto_increment primary key, name varchar(20) not null, stuNum int not null);

    create table students(id int auto_increment primary key, name varchar(20) not null, gender bit default 1, classid int not null, foreign key(classid)references class(id));

插入一些数据：insert into class values(0, “python1”, 55),(0, “python2”, 45),(0, “python3”, 52), (0, “python4”, 50);

insert into students values(0, "tom", 1 ,1);

关联查询：select students.name, class.name from class inner join students on class.id=students.classid;

分类：

  • 表A inner join 表B：

    说明：表A表B匹配的行会出现在结果集中

    示例：select students.name, class.name from class inner join students on class.id=students.classid;

  • 表A left join 表B：

    说明：表A表B匹配的行会出现在结果集中，外加表A中独有的数据，未对应的数据使用null填充

    示例：select students.name, class.name from class left join students on class.id=students.classid;

  • 表A right join 表B：

    说明：表A表B匹配的行会出现在结果集中，外加表B中独有的数据，对应的数据使用null填充

    示例：select students.name, class.name from class right join students on class.id=students.classid;

    • 文档更新

      • update()方法用于更新已存在的文档

        语法：db.集合名.update(

        ,

        ,

        {

    upset:<boolean>,

    multi:<boolean>,

    writeConcren:<document>

}

        )

        参数说明：

        query：update的查询条件，类似与sql里update语句内where后面的内容

update：update的对象和一些更新的操作符($set,$inc)等，$set直接更新，$inc在原有的基础上累加后更新

upset：可选，如果不存在update的记录，是否当新数据插入，true为插入，false    为不插入，默认为false

multi：可选，mongodb默认是false，只更新找到的第一条记录，如果这个参数为    true,就按照条件查找这个参数的数据全部更新

writeConcren：可选，抛出异常的级别

        示例：db.student.update({name:”tmo”},{$set:{age:25}})
